Reference implementation for the Honey Badger programming language.

val string_of_val : Defs.value -> Core.Std.String.t
Represent a value as a human-readable string.
val mul : Defs.value -> Defs.value -> Defs.value
Return a * b. Throws an exception in either a or b is a non-number.
val div : Defs.value -> Defs.value -> Defs.value
Return a / b. Throws an exception if either a or b is a non-number.
val add : Defs.value -> Defs.value -> Defs.value
Return a + b. If a and b are numbers, performs addition. If a and b are strings, concatenates them. If a and b are lists, concatenates them. Throws an exception otherwise.
val sub : Defs.value -> Defs.value -> Defs.value
Return a - b. Throws an exception if either a or b is a non-number.
val less : Defs.value -> Defs.value -> Defs.value
Return a < b. Throws an exception if either a or b is a non-number.
val cast_int : Defs.value -> Defs.value
casts v to an int. For ints, this returns v. For floats, this returns v rounded towards zero. For bools, true is 1 and false is 0. For strings, this tries to parse v as an int. Throws exceptions for other inputs or if v is a string that doesn't represent an int.
val cast_real : Defs.value -> Defs.value
casts v to a float. For ints, this returns v. For floats, this returns v. For bools, true is 1.0 and false is 0.0. For strings, this tries to parse v as a float. Throws exceptions for other inputs or if v is a string that doesn't represent a float.
val cast_bool : Defs.value -> Defs.value
casts v to a bool. For numbers, 0 is false and all others are true. For strings, "true" is true and "false" is false. For arrays and maps, empty is false, otherwise true. Throws exceptions for other inputs or if v is a string that is not "true" or "false".
val cast : Defs.value -> Defs.kind -> Defs.value
casts v to type t. For casting to int, see cast_int. For casting to float, see cast_float For casting to string, see string_of_val. Throws an exception for all others.
val eval : Defs.expr -> (string, Defs.value) Core.Std.Hashtbl.t -> Defs.value
Evaluates expr with the given state and returns a value.
val exec : Defs.expr -> Defs.value
Convenience function to wrap eval.
val main : string -> unit
Read source file from src, parse it as an expr, and print what it evaluates to.
